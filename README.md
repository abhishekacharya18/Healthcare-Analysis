# 🏥 Healthcare Data Analysis

---

## 📌 Project Overview

This project analyzes healthcare data to identify patient risk factors, track disease trends, and support better healthcare decisions. The goal is to extract insights for prevention, treatment, and resource planning.

---

## 🛠️ Tools Used

* 🐍 Python
* 📊 Pandas
* 📈 Matplotlib / Seaborn
* 📉 Excel
* 💾 CSV

---

## 📂 Dataset

* Healthcare dataset containing patient demographics, medical history, and health indicators
* Features include:

  * `age` 👶👵
  * `gender` 🚻
  * `bmi` ⚖️
  * `blood_pressure` 💓
  * `cholesterol_level` 🩸
  * `smoking_status` 🚬
  * `disease_outcome` ⚠️

---

## 🎯 Project Objective

* Explore patient demographics and health indicators
* Identify patterns related to disease risk
* Predict disease outcomes using machine learning
* Visualize key trends for better healthcare planning

---

## 🧹 Data Cleaning Steps

* Removed duplicate rows
* Handled missing values ❓
* Converted categorical variables 🔢
* Normalized numerical features 📏
* Split data into training and testing sets 🗂️

---

## 📊 Analysis Performed

* Distribution of patients by age, gender, and BMI 👥
* Correlation between health indicators and disease outcome 🌡️
* Impact of lifestyle factors like smoking on disease risk 🚬
* Visualized key trends using plots and heatmaps 📈🖼️

---

## 🤖 Model Building

* Logistic Regression
* Random Forest 🌲
* XGBoost ⚡
* Evaluated with Accuracy ✅, Precision 🎯, Recall 📡, F1-score ⚖️, ROC-AUC 🖤

---

## 📈 Key Insights

* Higher BMI and blood pressure increase disease risk ⚠️
* Smoking patients show higher incidence of disease 🚬
* Random Forest performed best with X% accuracy 🏆
* Certain age groups require focused monitoring 👶👵
